# How to clone this project:
### 1. Open terminal & change directory to save this project

### 2. Clone project
```
git clone https://github.com/rafirh/sitemaya.git
```
### 3. Change Directory
```
cd sitemaya
```
### 4. Install dependencies
```
composer install
```
### 5. Run Server
```
php artisan serve
```
### 6. Import sql
```
open localhost:8000 in your web browser
```
